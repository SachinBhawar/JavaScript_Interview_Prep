Object.keys()

Pure

Returns an array of a given object's own enumerable property names.

Parameters: obj (the object to extract keys from)
Returns: Array of property names

Use Case:

Iterating over object properties, getting property names

const person = {
  name: 'John',
  age: 30,
  city: 'New York'
};
const keys = Object.keys(person);
// ['name', 'age', 'city']

Object.values()

Pure

Returns an array of a given object's own enumerable property values.

Parameters: obj (the object to extract values from)
Returns: Array of property values

Use Case:

Getting all values from an object, data extraction

const person = {
  name: 'John',
  age: 30,
  city: 'New York'
};
const values = Object.values(person);
// ['John', 30, 'New York']

Object.entries()

Pure

Returns an array of a given object's own enumerable string-keyed property [key, value] pairs.

Parameters: obj (the object to extract entries from)
Returns: Array of [key, value] pairs

Use Case:

Converting objects to arrays, iterating over key-value pairs

const person = {
  name: 'John',
  age: 30
};
const entries = Object.entries(person);
// [['name', 'John'], ['age', 30]]

Object.assign()

Mutates

Copies the values of all enumerable own properties from one or more source objects to a target object.

Parameters: target, ...sources
Returns: The modified target object

Use Case:

Object cloning, merging objects, shallow copies

const target = { a: 1, b: 2 };
const source = { b: 4, c: 5 };
const result = Object.assign(target, source);
// { a: 1, b: 4, c: 5 }
// target is also modified

Object.create()

Pure

Creates a new object with the specified prototype object and properties.

Parameters: proto, propertiesObject (optional)
Returns: A new object with the specified prototype

Use Case:

Prototypal inheritance, creating objects with specific prototypes

const person = {
  isHuman: false,
  print: function() {
    console.log(`My name is ${this.name}`);
  }
};
const me = Object.create(person);
me.name = 'John'; // "name" is a property set on "me"

Object.defineProperty()

Mutates

Defines a new property directly on an object, or modifies an existing property.

Parameters: obj, prop, descriptor
Returns: The object that was passed to the function

Use Case:

Adding properties with special attributes (getters/setters)

const obj = {};
Object.defineProperty(obj, 'property', {
  value: 42,
  writable: false
});
obj.property = 77; // Throws an error in strict mode
console.log(obj.property); // 42

Object.freeze()

Mutates

Freezes an object: other code can't delete or change any properties.

Parameters: obj (the object to freeze)
Returns: The frozen object

Use Case:

Creating immutable objects, preventing accidental changes

const obj = {
  prop: 42
};
Object.freeze(obj);
obj.prop = 33; // Throws an error in strict mode
console.log(obj.prop); // 42

Object.seal()

Mutates

Seals an object, preventing new properties from being added and marking all existing properties as non-configurable.

Parameters: obj (the object to seal)
Returns: The sealed object

Use Case:

Restricting object modifications while allowing property updates

const obj = {
  prop: 42
};
Object.seal(obj);
obj.prop = 33; // Works
delete obj.prop; // Fails
obj.newProp = 'value'; // Fails

Object.getPrototypeOf()

Pure

Returns the prototype (i.e., the value of the internal [[Prototype]] property) of the specified object.

Parameters: obj (the object whose prototype is to be returned)
Returns: The prototype of the given object

Use Case:

Inspecting object prototypes, inheritance checks

const proto = {};
const obj = Object.create(proto);
Object.getPrototypeOf(obj) === proto; // true

const date = new Date();
Object.getPrototypeOf(date) === Date.prototype; // true

Object.setPrototypeOf()

Mutates

Sets the prototype (i.e., the internal [[Prototype]] property) of a specified object to another object.

Parameters: obj, prototype
Returns: The object with its prototype changed

Use Case:

Changing an object's prototype, dynamic inheritance

const animal = {
  makeSound() {
    console.log(this.sound);
  }
};
const dog = {
  sound: 'Woof!'
};
Object.setPrototypeOf(dog, animal);
dog.makeSound(); // 'Woof!'

Object.hasOwn()

Pure

Returns a boolean indicating whether the specified object has the indicated property as its own property.

Parameters: obj, prop
Returns: Boolean

Use Case:

Checking if a property exists directly on an object

const obj = {
  prop: 42
};
Object.hasOwn(obj, 'prop'); // true
Object.hasOwn(obj, 'toString'); // false

// Better alternative to obj.hasOwnProperty()

Object.getOwnPropertyNames()

Pure

Returns an array of all properties (including non-enumerable properties) found directly in a given object.

Parameters: obj (the object to inspect)
Returns: Array of property names

Use Case:

Getting all own properties of an object (including non-enumerable)

const obj = {
  a: 1,
  b: 2
};
Object.defineProperty(obj, 'c', {
  value: 3,
  enumerable: false
});
const props = Object.getOwnPropertyNames(obj);
// ['a', 'b', 'c']

Object.getOwnPropertyDescriptors()

Pure

Returns all own property descriptors of a given object.

Parameters: obj (the object to inspect)
Returns: Object containing property descriptors

Use Case:

Cloning objects with property attributes, detailed object inspection

const obj = {
  name: 'John'
};
const descriptors = Object.getOwnPropertyDescriptors(obj);
/*
{
  name: {
    value: 'John',
    writable: true,
    enumerable: true,
    configurable: true
  }
}
*/

Object.is()

Pure

Determines whether two values are the same value.

Parameters: value1, value2
Returns: Boolean

Use Case:

Strict equality checks, especially for edge cases

Object.is('foo', 'foo'); // true
Object.is(window, window); // true
Object.is(NaN, NaN); // true
Object.is(0, -0); // false

// Comparison with ===
NaN === NaN; // false
0 === -0; // true

Object.preventExtensions()

Mutates

Prevents new properties from ever being added to an object.

Parameters: obj (the object to make non-extensible)
Returns: The object being made non-extensible

Use Case:

Restricting object growth, creating fixed property sets

const obj = {
  prop: 42
};
Object.preventExtensions(obj);

// Can modify existing properties
obj.prop = 33; // Works

// Cannot add new properties
obj.newProp = 'value'; // Fails
